# Project instructions

## Environment

This project is developed locally in WSL2 (Ubuntu). Prefer WSL-native paths and commands for all workspace work.

- Run project tooling through WSL (for example, `npm`, `git`, and shell commands).
- Avoid Windows PowerShell and Windows command-line tools unless they are specifically required.
- Do not use Windows UNC paths as a working-directory workaround for project commands.
